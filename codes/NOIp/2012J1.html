Solution

This problem is very simple. You just need to find the small prime, and you'll get the big prime. So the solution will talk about how to check whether the integer is a prime.

We need to try all the integers from c{2} to o{⌊}o{√}t{v{n}}o{⌋}. Why o{√}t{v{n}}? That is because if there is a division v{d} which is bigger than o{√}t{v{n}}, then f{v{n}l{}v{d}} must be an integer, and it must be smaller than o{√}t{v{n}}. If both f{v{n}l{}v{d}} and v{d} are bigger than o{√}t{v{n}}, then f{v{d}o{}v{n}l{}v{d}}, which equals v{n}, must be bigger than o{(}o{√}t{v{n}}o{)}p{2}, which equals v{n} too. That is impossible. So you can check whether the integer v{n} is a prime by trying all the integers from c{2} to o{⌊}o{√}t{v{n}}o{⌋}.

P. S. In fact, we don't need to check whether the number is a prime. Because the problem have promised that v{n} will always be the product of two primes. So we just need to find the small number, and output their quotient.
