Solution

This problem is completely a mathematical problem that asked the number-theoretic reciprocal. We can calculate it with the extended Euclidean algorithm, which is also called exGCD.

exGCD is an algorithm which is able to find a pair of o{(}v{x}o{,}v{y}o{)} that satisfies the Bézout's identity v{a}o{}v{x}o{+}v{b}o{}v{y}o{=}o{gcd}o{(}v{a}o{,}v{b}o{)}. Obviously, we have v{a}o{}v{x}o{≡}c{1}o{(}o{mod}v{b}o{)}o{⇔}v{a}o{}v{x}o{+}v{b}o{}v{y}o{=}c{1}. Because the problem promised that it must have an answer, so o{gcd}o{(}v{a}o{,}v{b}o{)}o{=}c{1}, or there will be no number-theoretic reciprocal for this pair of o{(}v{a}o{,}v{b}o{)}. So the equation is a Bézout's identity.

The standard Euclidean algorithm proceeds by a succession of Euclidean divisions whose quotients are not used, only the remainders are kept. However, for the extended algorithm, the successive quotients are used. We keep the quotients and put it back into the equation, we will have v{x} and v{y}. But attention! v{x} and v{y} may be negative numbers. So you need to change it into the minimum positive number, that is in o{[}c{1}o{,}v{b}o{-}c{1}o{]}.
