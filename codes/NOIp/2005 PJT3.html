Solution

The problem is a 01 knapsack problem. So we can solve it by a dp algorithm.

Usually, a dp algorithm is made up of a dp array and the state transition formula. For a knapsack problem, the formula is dp[i]=maxj=tcost[i](dp[j-cost[i]]+price[i]). The use of it will be shown in the code below.

Code

Pascal

var
    dp:array[0..1000]of longint;
    t,m,cost,price,i,j:integer;
begin
    for i:=0 to 1000 do//Format the dp array
        dp[i]:=0;
    read(t,m);
    for i:=1 to m do
        begin
            read(cost,price);
            for j:=t downto cost do
                if dp[j]<dp[j-cost]+price then//dp=max(dp[i-cost]+price)
                    dp[j]:=dp[j-cost]+price;
        end;
    write(dp[t]);
end.

C++

#include<cstdio>
long dp[1001];
int main()
{
    for(int i=0;i<1001;i++)//Format the dp array
        dp[i]=0;
    int t,m;
    scanf("%d%d",&t,&m);
    for(int i=0;i<m;i++)
    {
        int cost,price;
        scanf("%d%d",&cost,&price);
        for(int j=t;j>=cost;j--)
            if(dp[j]<dp[j-cost]+price)//dp=max(dp[i-cost]+price)
                dp[j]=dp[j-cost]+price;
    }
    printf("%ld",dp[t]);
    return 0;
}
