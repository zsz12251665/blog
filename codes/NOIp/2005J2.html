Solution

Well, I know that the description looks that it requires a segment tree to solve. But it actually doesn't. It is just a simulation problem. You just need an array to keep whether there is a tree at the point and cut the ones in the interval down. In the end, count the number of the trees left.

Pascal

var `tree:array[0..10000]of boolean; `l,m,ans,s,e,i,j:integer; begin `read(l); `for i:=0 to l do//Initialize the tree array ``tree[i]:=true; `read(m); `for i:=1 to m do ``begin ```read(s,e); ```for j:=s to e do//Cut the trees down ````tree[j]:=false; ``end; `ans:=0; `for i:=0 to l do//Count the number of the trees ``if tree[i] then ```inc(ans); `write(ans); end.
