Problem

vjudge
HDU

Solution

This problem is a mathmatical problem. We can use the Vieta's formulas to solve the problem.

According to the Vieta's formulas, if a quadratic equartion which looks like x2+px+q=0 has two solutions of x, which may be x1 and x2, we'll have x1+x2=-p and x1x2=q. So in the problem we're given -p and q. According to the quadratic formula, a quadratic formula which looks like ax2+bx+c=0 have two solutions of x, they equal (-b+sqrt(b2-4ac))/2a or (-b-sqrt(b2-4ac))/2a. That means a quadratic equartion which looks like x2+px+q=0 with two solutions of x will have the solutions (-p+sqrt(p2-4q))/2 and (-p-sqrt(p2-4q))/2. So we can know that if sqrt(p2-4q) is an integer, the two solutions will be integers too. You ask why? If p is a odd number, p2-4q and sqrt(p2-4q) must be odd numbers too. So -p+sqrt(p2-4q) and -p-sqrt(p2-4q) must be even numbers, and the solutions are integers. If p is an even number, then p2-4q, sqrt(p2-4q), -p+sqrt(p2-4q) and -p-sqrt(p2-4q) must all be even numbers too. So the solutions are integers.

From last paragraph, we've known that we can judge if (p2-4q) is a square of an integer, the answer will be yes, or the answer will be no. So the problem is solved.
