Solution

This problem is solved in a clever way to use the union find set. Rather than each element, we maintain the relationship of the prefix sum so that the sum from the -th one to the -th one can be translated to the difference of the prefix sum to the -th and the prefix sum to the -th. Now you can use the weighed union find set to maintain the prefix sums.
