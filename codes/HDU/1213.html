Solution

This problem required a merge-find set.

The merge-find set is a data structure which can merge two disjoint sets and determine if two elements are in the same set. Each element has a pointer, which is its "father". So you can find the root of the set in O(lg(#n#)). At the same time, because it is a set, so we can put the pointer to the root after finding the root. It can reduce the time when you find the root of this element again. The time complexity of the merging is O(1). You just need to set the pointer of the root of the first set to the root of the second set.

After merging, we just need to scan all the elements and count the number of the roots to solve the problem.
