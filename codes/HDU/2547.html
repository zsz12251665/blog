Solution

Similar to HDU 2548, this problem is a mathmatical problem too. f(x,y,m,n)=√(x2+y2+m2+n2-2mx-2ny)=√((x-m)2+(y-n)2), which refers to the distance between (x,y) and (m,n). So when the hurt is reduced to mininum, the sum of the distances is reduced to minium too. It equals the distance between (a,b) and (c,d), which will be √((a-c)2+(b-d)2).

Code

Pascal

var
    t,i:integer;
    a,b,c,d:double;
begin//This problem does not need any comments to explain it
    read(t);
    for i:=1 to t do
        begin
            read(a,b,c,d);
            writeln(√((a-c)*(a-c)+(b-d)*(b-d)):0:1);
        end;
end.
