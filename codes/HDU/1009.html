Solution

The problem requires a greedy algorithm to solve. That can make each pound of cat food earn most JavaBeans. So As the first step, we should sort the rooms according to /#j#(_#i#_)/#f#(_#i#_)/. Then access room and took all JavaBeans we can take. In this way we can solve the problem.
