Solution

Well, first of all, we can get some information from the description. Firstly, your action on a handle have two stages: do and not do. That is because every twice you do on the same handle, it is as same as what it is like when you haven't do. Secondly, if you do on all the handles on the row i and column j, it will just change the stage of (i,j). That is because the handles which are not on row i or column j had been change twice. For example, (a,b) was change when doing on (i,b) and doing on (a,j). And the handles on the row i or column j(not include (i,j)) had been change 4 times(when doing on (i,1), (i,2), (i,3), (i,4) or when doing on (1,j), (2,j), (3,j), (4,j)). (i,j) had been change for 7 times(every change has covered on (i,j)), which is the only one that was changed odd times.

With the information above, the solution is very clear. First of all, scan the handles. Whenever finding a '+' on (i,j), do on every handle on the row i and column j. If it was done, then undo it. The information of whether it was done will be kept in the done array. After the scanning, count the number of the handles that are done. And then, print the number and the coordinates of the handles that are done.

Code

Pascal

var
    done:array[1..4,1..4]of boolean;
    i,j,k,ans:integer;
    handle:string;
begin
    for i:=1 to 4 do//Format the done array
        for j:=1 to 4 do
            done[i,j]:=false;
    for i:=1 to 4 do
        begin
            readln(handle);
            for j:=1 to 4 do
                if handle[j]='+' then//Scan the handles
                    begin
                        for k:=1 to 4 do
                            begin
                                done[i,k]:=not(done[i,k]);
                                done[k,j]:=not(done[k,j]);
                            end;
                        done[i,j]:=not(done[i,j]);//In the for-loop, (i,j) has been done twice, so add once more can keep that all the seven handles is done
                    end;
        end;
    ans:=0;
    for i:=1 to 4 do//Count the number
        for j:=1 to 4 do
            if done[i,j] then
                inc(ans);
    writeln(ans);
    for i:=1 to 4 do
        for j:=1 to 4 do
            if done[i,j] then
                writeln(i,' ',j);
end.

C++

#include<cstdio>
int done[4][4];
int main()
{
    char handle[4];
    for(int i=0;i<4;i++)//Format the done array
        for(int j=0;j<4;j++)
            done[i][j]=0;
    for(int i=0;i<4;i++)
    {
        scanf("%s",&handle);
        for(int j=0;j<4;j++)
            if(handle[j]=='+')//Scan the handles
            {
                for(int k=0;k<4;k++)
                {
                    done[i][k]=1-done[i][k];
                    done[k][j]=1-done[k][j];
                }
                done[i][j]=1-done[i][j];//In the for-loop, (i,j) has been done twice, so add once more can keep that all the seven handles is done
            }
    }
    int ans=0;
    for(int i=0;i<4;i++)//Count the number
        for(int j=0;j<4;j++)
            if(done[i][j]==1)
                ans++;
    printf("%d\n",ans);
    for(int i=0;i<4;i++)
        for(int j=0;j<4;j++)
            if(done[i][j]==1)
                printf("%d %d\n",i+1,j+1);//Don't forget to plus 1 because your loop is from 0 to 3 in C++
    return 0;
}
