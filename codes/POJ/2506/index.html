Solution

Let's solve the problem step by step. It is obvious that there is only one method to tile a rectangle(use no tiles) and a rectangle(use a tile). How about the number of methods for longer rectangles? Well, a method for a rectangle must have a standing tile, a tile or two lying tiles, right? If there is a standing tile, the number of methods will be the number of methods for a rectangle. If there is a tile or two lying tiles, the number of methods will be the number of methods for a rectangle. So the number of methods will be the number of methods for a rectangle plus twice the number of methods for a rectangle in total. The data range is lower than 250, so we can pretreat the numbers for every .
