Solution

Let's solve the problem step by step. It is obvious that there is only one method to tile a c{2}o{×}c{0} rectangle(use no tiles) and a c{2}o{×}c{1} rectangle(use a c{2}o{×}c{1} tile). How about the number of methods for longer rectangles? Well, a method for a c{2}o{×}v{i} rectangle must have a standing c{2}o{×}c{1} tile, a c{2}o{×}c{2} tile or two lying c{2}o{×}c{1} tiles, right? If there is a standing c{2}o{×}c{1} tile, the number of methods will be the number of methods for a c{2}o{×}o{(}v{i}o{-}c{1}o{)} rectangle. If there is a c{2}o{×}c{2} tile or two lying c{2}o{×}c{1} tiles, the number of methods will be the number of methods for a c{2}o{×}o{(}v{i}o{-}c{2}o{)} rectangle. So the number of methods will be the number of methods for a c{2}o{×}o{(}v{i}o{-}c{1}o{)} rectangle plus twice the number of methods for a c{2}o{×}o{(}v{i}o{-}c{2}o{)} rectangle in total. The data range is lower than 250, so we can pretreat the numbers for every v{i}.
