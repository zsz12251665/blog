Solution

At first, I wanted to learn about KMP. So I tried to solve this problem. But later I found that KMP is not necessary for this problem.

Let's mark a prefix of v{i} chars of the string as v{pre}b{v{i}}. If v{pre}b{v{i}} is a period of the string, we must have o{|}v{s}o{|}o{mod}v{i}o{=}c{0}, which o{|}v{s}o{|} is the length of the string v{s}. So we can enumerate the length of the period v{i}, and then scan the string to check whether the v{pre}b{v{i}} is a period of the string. So the time complexity is about o{O}o{(}o{|}v{s}o{|}o{∗}o{d}o{(}o{|}v{s}o{|}o{)}o{)}, which o{d}o{(}v{n}o{)} means the number of factors of v{n}.

By the way, the time for scaning the string once will not be as large as o{O}o{(}o{|}v{s}o{|}o{)} while solving the problem. This is because when you find that a prefix is not a period, you abandon it immediately so that you don't need to scan to the end of the string.
