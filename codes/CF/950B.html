Solution

It seems that the problem B is as difficult as the first problem in Senior Group, NOIp. The problem is asking how many consequent parts of the two arrays that are equal there are. So we can count the prefix sum of the two array and check if there are two numbers equal. What's more, as both #x#(_#i#_) and #y#(_#i#_) are bigger than zero, so the prefix must be raising up. So you can just add the number to the smaller prefix sum.
