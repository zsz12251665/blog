Solution

To minimize the maximum difference, we should find the maximum element and the minimum element next to the missing elements and set as the medium of them so that we can control their differences smaller than the half range of the elements between them. However, the maximum difference might be found between two fixed elements. So you should scan the whole sequence after replacing missing elements with to find the maximum difference as the answer.
