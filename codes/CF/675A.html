My brain must have been overheated for two months because I got 2 WA. That's a extremely simple problem. As you can see in the Code, if #a# equals #b#, the answer will be "YES" with no doubts. Otherwise, if #c# equals 0, there should be a "NO". Besides, if |#a#-#b#| is a multiple of |#c#|, and the sequence is incrasing (#c#>0 and #a#<#b#) or decrasing (#c#<0 and #a#>#b#), the answer will be "YES" too, or the answer will be "NO". We can use the ⊕ (xor) operator here, which calculates two boolean values like the table below:

#a# #b# #a# ⊕ #b#
True True False
True False True
False True True
False False False
