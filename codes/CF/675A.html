My brain must have been overheated for two months because I got 2 WA. That's a extremely simple problem. As you can see in the Code, if v{a}o{=}v{b}, the answer will be "YES" with no doubts. Otherwise, if v{c}o{=}c{0}, there should be "NO". Besides, if o{|}v{a}o{-}v{b}o{|} is a multiple of o{|}v{c}o{|}, and the sequence is incrasing (v{c}o{>}c{0} and v{a}o{<}v{b}) or decrasing (v{c}o{<}c{0} and v{a}o{>}v{b}), the answer will be "YES" too, or the answer will be "NO". We can use the o{⊕} (xor) operator here, which calculates two boolean values like the table below:

v{a} v{b} v{a}o{⊕}v{b}
c{True} c{True} c{False}
c{True} c{False} c{True}
c{False} c{True} c{True}
c{False} c{False} c{False}
