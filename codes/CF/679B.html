Solution

First of all, we find the maximum a that a3 is not greater than m. And the first block must have side a or(a-1). That is because if we use (a-2) or smaller integers. It must be smaller than using (a-1) because the first block can be replaced by a block with side (a-1). But why (a-1) cannot be replaced by a? Fox example, if we've an m=9, then a would be 2. But we know that the maximum number of the blocks is 7, with 13+13+13+13+13+13+13. So we may have more volumes to make a higher tower when we use the block with side (a-1).

Secondly, I have to remind you that when you pick up a block of with side a, the total volume of rest part should not be greater than ((a+1)3-a3), or this block is a illegal block because it is not the biggest block that can be taken. So when we choose to take the block with side a or (a-1), the volume remained are very important. The more volume remained, the higher tower we can build. So we should pick up the larger one.

After getting to know which block should we choose, we get the answer. After getting m, repeat choosing the most suitable block and remove it from the maximum volume or reset m until it is reduced to 0.
