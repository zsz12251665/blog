Solution

When I read the problem the first time, I thought out a way like solving the longest increasing subsequence problem, which means I just need an array to keep the longest subsequence until this integer. The code is given as Solution 1 in the code part.

However, when I submitted it to the judge, the answer was TLE because the time complexity is o{O}o{(}v{n}p{2}o{)}. Later, I got another algorithm whose time complexity is o{O}o{(}v{n}o{}o{lg}v{a}b{v{i}}o{)}. It just needs a dp algorithm. v{dp}b{v{i}} keeps the length of the longest subsequence whose last integer's v{i}-th digit is not 0 in binary. So after read in each integer in sequence, I can update the v{dp} array in an o{O}o{(}o{lg}v{a}b{v{i}}o{)} algorithm. And that is Solution 2, which is accepted.
