Solution

When I read the problem the first time, I thought out a way like solving the longest increasing subsequence problem, which means I just need an array to keep the longest subsequence until this integer. However, when I submitted it to the online judge, the answer was TLE because the time complexity is (You enumerate the elements along the sequence and check if the longest subsequence that ends with it can update the longest subsequence that ends with any element after it). Later, I got an acceptable algorithm whose time complexity is . It just needs a dp algorithm, which keeps the length of the longest subsequence whose last integer's -th digit is not 0 in binary. So after reading an integer in sequence, we can find a longest subsequence that ends with it and update the array with this subsequence in an algorithm.
