This problem is not difficult. If you observe carefully, you'll find that the numbers in the same row have the same difference in the same line, which means #M#(_#i#,#j#_)-#M#(_#i#+#a#,#j#_)=#M#(_#i#,#j#+#b#_)-#M#(_#i#+#a#,#j#+#b#_). That is the way to solve the problem.
