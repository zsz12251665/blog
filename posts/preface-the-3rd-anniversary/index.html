Preface: the 3rd Anniversary

2020-01-24 22:53 by zsz12251665

Happy Chinese New Year! My little blog has come through 3 years! In 2019, zsz12251665's blog changed a lot. Now it is time to review it.

The domain

I am terribly sorry that I forgot to renew the free domain "zsz12251665.cf" and I would need to pay for it after it expired. Noticing that Wang Huaijie has hosted his blog on ZEIT, I turn to it too. In the past, I deployed my page on GitHub Pages, using free domains from Freenom and enabled HTTPS and CDN via CloudFlare. But now ZEIT provides a free deploying plan with HTTPS-enabled custom domains, continuous deployment with GitHub and high-performance Smart CDN. So I don't need to spend as much time to deploy my blog as I did before.

The UI

You might noticed that our UI also got updated. We abandon the navigator bar and use little icons instead. We also separate the settings to a single page.

Homepage Comparison

Inspired by the Metro UI from Microsoft, I dropped the card design from the Material Design because it did not seem like a card at all. The big plain design of the header and the footer makes the UI not as complex and weird as the large and conspicuous header and the tiny and unnoticeable footer.

However, there are some problems here. The link colour is similar to the text colour. So some letters and icons at the header and footer might be unreadable in some themes.
