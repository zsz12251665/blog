Pseudo Active

As you know, my blog is hosted by GitHub Pages or Coding Pages. Pages service is only available for static pages. So the interactivity is poor and the content is usually mixed with the template. So I managed to find a way. In 2017, the content is put under the body element and a piece of JavaScript will load the template and put the content in it. In 2018, we developed to use AJAX. Now the content was seperated in files as HTML pieces. And the JavaScript will pick them up and put it into the template by himself. You may probably find it. We are including searching parameters in URLs. They content the key information to the contents. So much fewer pages was used now. It works similarly to Active Pages: getting a request, finding the contents in the database and sending them back with the template. The difference is my contents are saved in files. So I suppose my blog is pseudo active.
